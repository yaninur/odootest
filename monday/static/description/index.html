FEATURES

Allows you to restrict Sale Order emails/notifications sent out from any particular sales team account

Just a simple checkbox to execute this module

DESCRIPTION

This Module was developed with the objective to reduce the amount of auto-emails your vendor will receive when a sales order is created, amended, confirmed and revised. You can still send out an email manually (once) to your vendor when you are completely sure of issuing the SO.

HOW IT WORKS

Install the module, it adds a checkbox in Sales Team configuration page

If the box is checked, there will be no emails going out when a particular sales team confirms a sale order
